# authorize
🔒 **API de Autorização**: Uma API simples para verificar a autorização de acesso.

ℹ️ **Endpoint**: `/v1/authorize`

🔑 **Resposta JSON**:
```json
{
  "status": "success",
  "data": {
    "authorization": true
  }
}
```
🔄 **Pode conter uma surpresa!**

🌐 **Domínio para teste: O domínio de teste pode estar ou não funcionando.**
